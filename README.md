# GenIA

---

## Requirements

- node >= v16.16.0
- npm or yarn

## Built with

- ViteJS (React)
- ChakraUI
- React Query
- Zustand
- OpenAI API

## Getting Started

1. Clone the project


2. Install dependencies

```
npm install
```

or

```
yarn install
```

3. Environment (important)
   Create an .env file at the root of the project with the following key:

```
//.env
API_LINK = your own GenIA
```
